El algoritmo no premia lo bonito. Premia lo rápido y estructurado. ⏱️

Puedes tener el diseño más espectacular del mundo, pero si tu servidor es lento, las imágenes no están comprimidas y tu código bloquea el renderizado, estás invisible para los motores de búsqueda.

## ¿Qué son los Core Web Vitals?

Son las métricas oficiales de Google para medir la experiencia del usuario real:
1.  **LCP (Largest Contentful Paint):** Cuánto tarda en cargar el elemento más grande de tu pantalla.
2.  **INP (Interaction to Next Paint):** Cuánto tarda tu sitio en reaccionar cuando el usuario hace clic.
3.  **CLS (Cumulative Layout Shift):** Cuánto "saltan" los elementos mientras se carga la página.

### Escribimos código, no arrastramos bloques
